# fssh

fssh (fast ssh) is a bash script that manage your ssh connection.

## Installation

Just copy it into your /usr/bin and chmod +x it and boom.

```bash
sudo cp fssh /usr/bin/fssh && sudo chmod +x /usr/bin/fssh
```

## Usage

```bash
┌─[ fssh ] on  main [?] took 3,2s 
➜ fssh yx
1) pyx
2) jyx
3) server-9yx
#? 
```
